：多维度（玩法、交互设计、视觉美术、题材）评估产品
玩法可以分为创新性、耐玩性和契合度三个维度：
1.创新性
，顾名思义，就是这款产品跟别的产品有什么不一样的地方，做了哪些突破；
2.耐玩性
，对一款产品来说，至少要能够保证用户在其中数十小时的正常游戏体验；
3.契合度
，是玩法和题材之间的匹配程度。因为现在的游戏行业，尤其国内游戏行业非常讲究IP，游戏一定有IP加成，但从实践来看，不是所有IP都适合所有玩法，这里其实存在配合度和契合度的问题
视觉美术也分成了辨识度、美术质量、接受程度三个维度：1.辨识度
，主要指这个产品的第一印象，是否能抓住人的眼球，是否能和别的产品有所区别，主要是美术风格的区别。比如《女神异闻录5》，它是一个日式RPG，这款产品的制作预算不是特别高，在3A级产品中肯定不是顶尖的。它美术的质量，比如模型、贴图这种
纯制作
的东西的精度，比起《荒野大镖客》要差很多，但是这款产品的美术效果却出乎意料地让人很满意。这主要得益于，它采用了一种比较特殊的美术风格，时尚、潮流、现代的美术风格，保证了很高的辨识度；
2.美术质量
，这个比的就是制作，就是纯粹的美术质量；
3.接受度

模块
二
：核心乐趣提炼与品类定位 (Core Fun &amp; Genre Positioning)
[ ] 
核心乐趣提炼：
 提炼该游戏最吸引人的 3-4 
个
核心乐趣点。这些乐趣点是玩家持续投入时间的根本驱动力。 
[ ] 
真实品类定位：
 基于核心乐趣点反推该产品的真实所属品类（而非表面的商店分类标签）。例如：《皇室战争》表面是“卡牌游戏”，但核心乐趣（实时对抗 + 临场指挥）决定了它本质上是一款 RTS。 
[ ] 
混合品类解构：
 识别该产品的“混合品类”构成及各品类元素的权重占比。例如：《原神》= 开放世界探索（40%）× 二次元角色收集（30%）× 元素反应战斗（20%）× 剧情叙事（10%）。明确哪个品类基因是“主驱动”，哪些是“辅助增强”。
模块二：品类基准比较 (Genre Benchmark Analysis)
[ ] 
品类通用特性归纳：
 归纳该产品所属品类的通用优势与通用痛点。可通过同类型产品横向比较（寻找共同乐趣点）、相近类型产品纵向比较（寻找差异点）、以及现实映射关系（理解品类的底层体验诉求）来提炼。 
[ ] 
品类基因四维分析：
 将该产品与其所属品类的通用特性进行对比，从四个维度进行分析： 
继承 (Inherit)：
 保留了品类的哪些核心体验？这些继承是否忠实还原了品类的核心乐趣？
强化 (Strengthen)：
 在哪些方面做到了品类最佳或显著超越同类产品？强化的手段是什么（技术突破、设计创新、资源堆砌）？
创造 (Create)：
 引入了哪些品类中前所未有的新元素？这些创新是否与品类核心体验产生了正向化学反应？
解决 (Solve)：
 解决了该品类长期存在的哪些痛点？解决方案是否引入了新的问题？

[ ] 
品类生命周期定位：
 判断该品类当前处于生命周期的哪个阶段（探索期 / 成长期 / 成熟期 / 变革期），并评估该产品在品类周期中的入场时机是否合理。 
[ ] 
产品生态位判断：
 评估该产品在品类演化趋势中的位置：是顺应品类主流趋势的“正统继承者”，还是试图开辟新方向的“变革挑战者”？其生态位选择是否与自身的资源禀赋（团队能力、资金规模、IP 积累）相匹配？
